# Mini-CEO: CrossSell Optimizer

## О проекте
Платформа анализирует профиль клиента и подбирает оптимальный продукт и канал коммуникации, минимизируя затраты на привлечение (CPA) и максимизируя инкрементальный доход (Uplift).

**Основные функции:**
- **Uplift-моделирование:** Отсеивание сегментов «Спящие собаки» и «Потерянные клиенты».
- **Максимизация EV:** Выбор оффера на основе формулы `(Uplift * LTV) - CPA`.
- **Интеллектуальная маршрутизация:** Автоматический выбор канала (Push, СМС, Звонок).

## Технологический стек
- **Backend:** Python (FastAPI) — высокопроизводительный API для ML-расчетов.
- **Frontend:** HTML5, CSS3 (Modern Glassmorphism Design), JavaScript (ES6+).
- **ML Logic:** Собственные модели склонности (NBP) и инкрементальности (Uplift).
- **Deployment:** Render (Backend), GitHub Pages (Frontend).

## Алгоритм принятия решений
Сердце системы — `DecisionEngine`, который использует данные из четырех ML-моделей:
1. **NBP (Next Best Product):** Вероятность покупки конкретного продукта.
2. **Uplift Model:** Сегментирует базу, отсекая тех, кто купит сам, и тех, кого нельзя беспокоить.
3. **NBC (Next Best Channel):** Выбирает самый дешевый из эффективных каналов, следуя правилу каскада (Waterfall).
4. **MTA Attribution:** Пересчитывает реальный вклад каналов для обучения системы на основе iCPA , а не простого факта продажи.
